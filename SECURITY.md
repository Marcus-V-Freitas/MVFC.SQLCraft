# Security Policy

## Supported Versions

Only the latest released version receives security fixes.

| Version | Supported |
|---------|-----------|
| Latest  | ✅        |
| Older   | ❌        |

## Reporting a Vulnerability

**Do not open a public GitHub Issue for security vulnerabilities.**

Please report using [GitHub Security Advisories](https://github.com/Marcus-V-Freitas/MVFC.SQLCraft/security/advisories/new).

You can expect an acknowledgement within **5 business days**.

## Scope

Security concerns for this project typically include:

- Vulnerable NuGet package dependencies
- Insecure defaults in container configurations exposed to the host
- Credential handling in helper constructors or configuration methods
